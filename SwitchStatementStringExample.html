Switch Statements with Strings Example

This is an alternative to if else statements with a string. It is better to use a Switch Statement when you have nested if else Statements or lots of if else statements that make understanding the code on the page difficult.



Take a note and remember that JavaScript is case sensitive. Any string that the customer enters must match exactly for the Switch Statement to work correctly. So if the user types in capital letter first either list it or use another script to convert all the characters in the string to the same case. In the example below the friut may be typed in with a plural or without and it will still be recognized by the javascript.
